The models for the paper:

PLoS Comput Biol 14:e1006535[PubMed]

are available in the supplementary links of the paper:

https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1006535#sec034

and the authors github repositories:

https://github.com/AllenInstitute/arkhipov2018_layer4

which used BioNET (Python interface to NEURON) which is available here

https://alleninstitute.github.io/bmtk/bionet.html

Here is a link suggested by Anton Arkhipov:
